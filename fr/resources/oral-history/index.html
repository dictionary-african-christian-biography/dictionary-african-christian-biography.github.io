Recherche en histoire orale - Documents pédagogiques

Vous trouverez ici des documents et fiches utiles pour vos projets de biographie et/ou d’histoire orale. Pour les directives générales de soumission d’une biographie au DIBICA, voir cette page.

  1. Programme complet de l’atelier d’histoire orale (PDF): “Faire de l’histoire orale: aider les chrétiens à raconter leur propre histoire” – Programme de cours basé sur le travail original de Prof. Jean-Paul Wiest (2010).
  2. Fiche planification d’un projet d’histoire orale (Version Word); Version PDF – Ce document vous aidera à planifier votre projet de biographie.
  3. Modèle de fiche explicative(Version Word); Version PDF – un document que vous pouvez adapter pour expliquer votre projet de recherche aux interviewés potentiels.
  4. Modèle de lettre d’autorisation de recherche(Version Word); Version PDF – un document à utiliser par un institut/université pour expliquer les buts de la recherche aux interviewés potentiels.
  5. Maquette pour la collecte d’informations (Version Word); Version PDF – une fiche qui vous permet de ne rien oublier dans votre collecte de données pour une biographie.
  6. Procédures d’interview(Version Word); Version PDF – une liste de choses à faire avant, pendant, et après l’interview.
  7. Techniques d’interview(Version Word); Version PDF – exemples de bonnes et de mauvaises questions d’interview.
  8. Analyse contextuelle(Version Word); Version PDF – une fiche pour récolter des informations clés en lien avec la préservation de l’histoire d’une église ou dénomination.
  9. Ressources utiles pour la recherche sur le sujet du christianisme africain: Version Word; Version PDF
